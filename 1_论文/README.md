# 论文正文提交说明

本目录用于存放您的期末论文。

### **提交要求：**
1. **格式要求**：请务必使用 `0_Guide_Materials/大学物理期刊论文模板.doc` 进行排版。
2. **提交文件**：请在本目录下上传您的论文最终版（推荐 **PDF** 格式，文件名请命名为 `论文正文.pdf`）。
3. **图片管理**：论文中使用的所有插图源文件（如 Matplotlib 生成的 `.png` 或 `.pdf`）请统一存放在本目录下的 `assets/` 文件夹中。

---
*注：如果您坚持使用 Markdown 撰写，请确保最终导出的 PDF 符合期刊模板的排版要求。*
