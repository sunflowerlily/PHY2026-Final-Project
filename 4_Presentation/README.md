# 现场答辩 (Presentation) 准备说明

本目录用于存放您的现场汇报材料。

### **提交要求：**
1. **演示文稿**：请上传您的现场汇报 PPT（推荐 **.pptx** 或 **.pdf** 格式，文件名请命名为 `[论文题目]_组名.pptx`）。
2. **辅助材料**：若有模拟动画视频、交互式程序演示等，请统一存放在本目录下。

### **汇报安排：**
- **汇报时间**：第 18 周现场（具体时间见课程公告）。
- **时长要求**：每组 **8 分钟汇报 + 2 分钟 Q&A**。
- **重点内容**：
    - 物理问题背景与核心方程。
    - 数值算法的实现与**稳定性/精度验证**。
    - 模拟结果的**物理意义分析**。
    - 团队分工与 AI 使用心得。
