# 3_Data/ - 模拟数据目录 (可选)

**目的**: 存放模拟过程中产生的数据或用于分析的外部公开数据。

### **管理原则：**
1.  **按需存放**: 若模拟生成的数据量较小（如几百 KB 的 CSV/JSON），建议上传以方便复现结果。
2.  **大文件禁令**: 若单个数据文件超过 **50MB**，请勿上传至 GitHub。您可以在此说明如何重新生成或获取这些数据。
3.  **公开数据**: 若项目基于天文学公开数据集，建议在此存放一个“微缩采样版”供演示使用。

### **目录说明：**
- `raw_data/` (可选): 存放原始输出数据。
- `processed_data/` (可选): 存放经分析脚本处理后的汇总数据。
