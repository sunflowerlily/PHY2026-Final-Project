## 模块 A：ODE 数值积分与经典动力系统（第 6–9 周）

本模块对应课程中 Euler、RK4、Verlet/Leapfrog 的学习内容。项目需要学生自行选择积分器，分析步长对能量守恒的影响，以及（可选）AI 生成代码的对比审查。

### 选题 A-1：Lorenz 系统与混沌——RK4 实现与相图可视化

**核心物理**：大气对流的 Lorenz 三维 ODE 系统是混沌行为的经典案例，具有极强的可视化张力。学生用 RK4 积分三个耦合方程，研究初始条件敏感性（蝴蝶效应），并绘制 3D 相图动画。

**可选拓展**：比较 Euler 与 RK4 在不同步长 $\Delta t$ 下的轨道分叉行为，分析数值误差对混沌系统的放大效应。

**参考文献与代码起点**
- Lorenz 吸引子 RK4 Python 3D 动画实现（GitHub: vaibhawvipul/lorenz-attractor-simulation）[^3]
- 技术博客：Solving the Lorenz system using Runge-Kutta methods（Fabrizio Musacchio）[^4]
- GitHub: ShrirajHegde/lorenz_system-rk4（Fortran+Gnuplot 版本，可用于对比）[^5]

**AI 融合点**：让 AI 生成初版 RK4 Lorenz 代码，学生负责审查能量/体积收缩守恒是否正确，并在报告中说明修改了哪些错误。

**可选拓展**：加入第三个天体（如木星），研究共振或 Kozai–Lidov 机制的简化模型。

**参考文献与代码起点**

### 选题 A-3：Lane-Emden 方程——多方球恒星结构的数值积分

**核心物理**：Lane-Emden 方程描述多方球（polytropic）近似下的恒星静压平衡结构。用 RK4 或 shooting method 求解不同多方指数 $n$ 下的密度剖面，并与解析解（$n=0,1,5$）对比。

**可选拓展**：计算各 $n$ 对应的星体总质量、半径与中心密度的关系；与标准太阳模型对比 $n=3$ 多方解。

## 模块 B：PDE 数值方法（第 10–12 周）

本模块对应 FTCS、Crank-Nicolson、SOR 等有限差分方法。项目应当展示离散化过程、边界条件处理和 CFL 稳定性分析。

### 选题 B-1：热传导方程的 Crank-Nicolson 求解与稳定性分析

**核心物理**：一维或二维热传导（扩散）方程 $\partial_t u = \alpha \nabla^2 u$ 是抛物型 PDE 的标准模型。使用 Crank-Nicolson 格式（半隐式，时间二阶精度）求解，对比 FTCS 的稳定性条件。

**可选拓展**：将边界条件改为非均匀/周期性；研究 CFL 数对两种格式精度的影响；或求解二维版本。

## 模块 F：物理信息神经网络 PINN（第 16 周）

本模块对应课程最前沿的 AI for Physics 内容。项目的评分重点在于：学生能否理解 physics loss 的物理含义，并与解析解/有限差分解做定量对比，而不仅仅是跑通代码。

### 选题 F-1：用 PINN 求解阻尼/驱动简谐振子 ODE

**核心物理**：构造含物理约束的损失函数

$$ \mathcal{L} = \mathcal{L}_{\text{data}} + \lambda \mathcal{L}_{\text{physics}}, $$

其中物理项来自阻尼振子方程 $\ddot{x} + 2\zeta\omega_0 \dot{x} + \omega_0^2 x = F(t)$。改变阻尼系数 $\zeta$，对比 PINN 预测与解析解的相对误差，分析 $\lambda$ 权重对收敛的影响。

**可选拓展**：拓展到简单摆（含非线性项 $\sin\theta$）；或做"逆问题"——给定噪声观测数据，用 PINN 反推阻尼系数 $\zeta$。

**参考文献与代码起点**

### 选题 G-3：引力波参数推断——从 GW250114 事件中破解黑洞视界面

**核心物理**：在最近的一篇文章 [[Nature, 2026](https://www.nature.com/articles/s41586-026-10696-0)] 中，作者声称在引力波事件 GW250114 中发现了黑洞视界面的“参考系拖拽效应”，这打开了深入理解黑洞视界面的一扇窗户。好消息是该事件对应的引力波信号在网站上可以公开获取，方便我们验证他们的说法。事实上使用 MCMC 处理引力波的数据并不困难，只需要理解三个简单的要点并遵循分析的五步骤就可以完成。

**要点解析**：
1. **数据本质**：探测器测到的引力波数据只是一个一维时间序列（表征激光探测器的光腔长度被引力波压缩的比例），每个探测器分别会测到一个时间序列。
2. **参数依赖**：理所当然地，每个探测器的光腔长度被压缩的比例主要取决于下面两组**参数**：1) 几何参数：探测器的空间方位、取向与源事件的空间方位；2）物理参数：源的本征（源参考系）物理性质。
3. **似然函数构建**：构建引力波数据分析的**似然函数**与课堂上讲的带误差的数据拟合的例子逻辑是一样的，只是要转换思维——在频率空间处理问题——而已，即假设每个频率分量的残差（即物理波形的傅里叶变换与探测到的数据的傅里叶变换之差）服从独立高斯分布（详细公式见 [[arXiv:1811.02042](https://arxiv.org/abs/1811.02042), 公式 4.1]）。

**要求**：
使用软件 Bilby [[官网](https://bilby-dev.github.io/bilby/)] 分析双黑洞并合的引力波事件 GW250114 的并合产物的质量和自旋：
1. 与参考文章作对比；
2. 解释其中的物理意义。

**分析五步骤** [参考 Bilby 官网实例：[GW150914.py](https://github.com/bilby-dev/bilby/blob/main/examples/gw_examples/data_examples/GW150914.py)]：
1. **数据下载**：在公开数据官网 [[GWOSC](https://gwosc.org/eventapi/html/GWTC-5.0/GW250114_082203/v2/)] 下载原始数据，包括两个探测器（H1 和 V1）的数据；
2. **噪声分析**：分析所选数据段的噪声特性（PSD，回顾讲义傅里叶分析部分）；
3. **波形选择**：选择适当的波形模版（目前人类的解析计算能力还不能完整算出引力波波形，很多波形是各种近似计算和模拟波形的糅合，没有绝对的“最准”，只有“最适合”）；
4. **推断求解**：构建数据依赖的似然函数（取决于数据 PSD 和波形模板），运行 MCMC；
5. **物理推断**：分析参数分布，推断物理信息（注意引力红移导致地球参考系和源参考系的质量不同，注意转化）。
